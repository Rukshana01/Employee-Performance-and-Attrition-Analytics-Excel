# Employee-Performance-and-Attrition-Analytics-Excel
Interactive HR Analytics Dashboard built in Microsoft Excel using PivotTables, Charts, KPIs, and Slicers.

Project Overview:
This project analyzes employee data using Microsoft Excel to identify workforce trends, salary distribution, and employee attrition patterns. The objective is to transform raw HR data into meaningful insights through PivotTables, charts, KPIs, and an interactive dashboard that supports HR decision-making.

Objectives:
* Analyze employee attrition by department and gender.
* Study salary distribution across departments.
* Categorize employees by age groups.
* Build an interactive HR dashboard.
* Provide key HR metrics for better decision-making.

Dataset:
The dataset contains **100 employee records** with the following fields:
   * Employee ID
   * Age
   * Gender
   * Department
   * Salary
   * Attrition Status

Project Workflow:
1. Data Preparation
* Imported the HR dataset into Excel.
* Checked for missing values and duplicate records.
* Corrected inconsistent data types.

2. Data Analysis
* Created PivotTables for department-wise and gender-wise attrition.
* Analyzed salary distribution across departments.
* Grouped employees into age categories.

3. Data Visualization
Created charts to visualize:
  * Department-wise Attrition
  * Gender-wise Attrition
  * Salary Distribution
  * Age Distribution

4. KPI Metrics
Calculated:
* Total Employees
* Employees Left
* Attrition Rate
* Average Salary

5. Interactive Dashboard
Developed an Excel dashboard featuring:
* KPI Cards
* Pivot Charts
* Department Slicer
* Gender Slicer

Key Insights:
* Identified departments with higher attrition.
* Compared attrition trends between male and female employees.
* Analyzed salary variations across departments.
* Examined employee distribution across different age groups.
* Created an interactive dashboard for quick HR reporting.

Skills Demonstrated:
* Data Cleaning
* Data Analysis
* PivotTables
* Pivot Charts
* Excel Functions (COUNTIF, COUNTA, AVERAGE, IF)
* Dashboard Development
* Data Visualization
* Business Reporting

Project Outcome:

This project demonstrates how Microsoft Excel can be used to perform HR analytics and create professional dashboards. It showcases practical skills in data analysis, reporting, and visualization, enabling HR professionals to make informed, data-driven decisions.
